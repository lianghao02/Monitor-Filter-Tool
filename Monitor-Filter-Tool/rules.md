# 📄 卷宗二：rules.md (時空重連與防遮蔽憲法細則)

本卷宗紀載 AG-Forensic-Player 專案的核心演算法參數與防護憲法，為確保數位鑑識之公信力與有效性，以下指標已列為系統底層鐵律，嚴禁任何未經 Tech Lead 授權的篡改。

## 1. 時空臨界盲區重連機制 (Time-Space Reconnection)
- **1.5 秒影片時戳冷卻寬限期**：為對抗監視器常見的「掉幀」、「物體被短暫遮蔽（如被電線桿擋住）」導致追蹤 ID 斷裂的問題，系統內建 1.5 秒的時空重連緩衝期。
- 若嫌犯短暫消失於畫面中，追蹤器不會立刻判定「離開」，而是會在記憶體中保留該目標的狀態與歷史最佳特徵格。只有當目標消失超過 1.5 秒，系統才會判定其已徹底脫離監視範圍，並將收集到的最清晰特徵格寫入硬碟結案。

## 2. ROI 綠色防線自適應感測器 (ROI Adaptive Sensor)
- **80px ~ 200px 的 Min-Max 自適應半徑鎖**：傳統寫死的 ROI 觸發區域容易因為攝影機角度不同而失靈。本系統導入自適應觸發半徑，最低靈敏度鎖定於 80px，最高不超過 200px。
- 當嫌犯的重心座標（Centroid）踏入此動態浮動半徑內，系統即精準發動「進出瞬間捕捉」快門，徹底防範誤觸發與漏拍。

## 3. 鑑識視覺無干擾化 (Forensic Visual Governance)
- **1 像素 (1px) 極細標記框**：鑑識實務上，粗糙的 Bounding Box 極易遮擋嫌犯手上的關鍵細小證物（如毒品夾鏈袋、凶器刀柄）。
- 全系統不論是 YOLO 偵測框或是 ROI 防線，皆強制鎖死在 1px 寬度，確保畫面資訊 100% 穿透無死角。

## 4. 時空軌跡防偽壓印 (Timecode Watermarking)
- **Bottom-Center（正中央最底部）防偽時戳浮水印規則**：部分監視器原生已自帶左上或右上的時間浮水印。為了防止我們壓印的 OCR 時間軸與原始畫面資訊互相覆蓋、引發法庭證據效力爭議，AG-MONITOR 的時間壓印鐵律鎖定在畫面的正下方置中（Bottom-Center）。
- 字體採用抗鋸齒的高對比配色，確保每一幀數位證物皆具備無可辯駁的時間公信力。
